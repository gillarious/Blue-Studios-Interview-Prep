# Arrays

### Implementation

```python=1
arr = [0, 1, 2, 3, 4, 5]
```

### Runtime Analysis

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| index            | O(1)             |
| index assignment | O(1)             |

```python=1
arr = [0, 1, 2, 3, 4, 5]
arr[1]
arr[2] = 7
```

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| length           | O(1)             |

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| append           | O(1)             | 

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| pop() or pop(-1) | O(1)             |
| pop(i)           | O(N)             |
