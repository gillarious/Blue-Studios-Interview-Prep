# Arrays

### Implementation

```python
arr = [0, 1, 2, 3, 4, 5]
```

### Runtime Analysis

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| index            | O(1)             |
| index assignment | O(1)             |

```python
arr[1] 
# returns 1

arr[2] = 7 
# arr = [0, 1, 7, 3, 4, 5]
```

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| length           | O(1)             |

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| append           | O(1)             | 

| Function         | Big O Complexity |
| ---------------- | ---------------- |
| pop() or pop(-1) | O(1)             |
| pop(i)           | O(N)             |
